```

```

**بتطلب منك:**

الملف اللي اتعمل ليك: `data-studio-ai.html`

**اختر واحد من الخيارات:**

**أ) تحميل الملف من هنا:**

**ب) نسخ الملف:**
- أي ملف HTML (أي حد)

---

### **الخطوة 4: ارفع الملف على Vercel**

بعد الدخول إلى Vercel:

```
1. اضغط: "New Project"
2. اختر: "Upload"
3. اختر الملف: data-studio-ai.html
4. اضغط: Upload
5. انتظر 30 ثانية
6. بتطلع رابط مثل:
   https://your-project.vercel.app
```

## 🎬 **مثال عملي كامل:**

### **الخطوة 1: افتح التطبيق**
```
https://your-project.vercel.app
```

### **الخطوة 2: حمّل بيانات تجريبية**
```
اضغط: 🏪 بيانات مبيعات
┌─────────────────────────┐
│ "تم تحميل البيانات" ✅  │
└─────────────────────────┘
```

### **الخطوة 3: اذهب إلى البيانات**
```
اضغط التبويب: 📊 البيانات
بتشوف جدول البيانات الكاملة
```

### **الخطوة 4: ابحث عن شيء**
```
في البحث، اكتب: "لابتوب"
الجدول بيقفز للنتائج المطابقة
```

### **الخطوة 5: نظّف البيانات**
```
اضغط: ✨ التنظيف
اختر: ☑ كل الخيارات
اضغط: [تشغيل التنظيف]
انتظر 2 ثانية
البيانات نظيفة! ✅
```

### **الخطوة 6: حلّل البيانات**
```
اضغط: 📈 التحليل
شوف الإحصاءات والرسوم البيانية
```

### **الخطوة 7: اسأل AI**
```
اضغط: 🤖 تحليل AI
اكتب: "ما هي أفضل منطقة للمبيعات؟"
اضغط: [أرسل]
Claude يجاوب! 🧠
```

### **الخطوة 8: صدّر البيانات**
```
اضغط الزر الأزرق: [تصدير CSV]
الملف ينزل في Downloads
```

## 📞 **لو حصلت مشكلة:**

### **المشكلة: "الصفحة ما تفتح"**
```
الحل:
1. تأكد من الإنترنت
2. اكتب الرابط صح
3. جرّب متصفح تاني
```

### **المشكلة: "الزر ما يشتغل"**
```
الحل:
1. جرّب إعادة تحميل الصفحة (F5)
2. امسح الـ Cache: Settings → History → Clear
3. جرّب متصفح جديد
```

### **المشكلة: "ما أعرف أستخدم AI"**
```
الحل:
1. اذهب لـ "تحليل AI"
2. في خانة الكتابة، اكتب سؤالك
3. مثال: "كم عدد المنتجات المباعة؟"
4. اضغط [أرسل]
5. Claude يرد! 🧠
```
